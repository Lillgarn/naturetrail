# naturetrail

Webbsidor, ljudfiler och qr-koder för Vässarös naturspår "Naturnäran".

## För att bidra:

1. I en python-venv, installera mkdocs, mkdocs-material, mkdocs-glightbox och mkdocs-audio. 
2. Gör ändringar i `/src/docs/` och kör sen `mkdocs build` i `/src/`.

Kommandot `mkdocs serve` ger en live preview av sidan.
